# Game-Random-Number
It is simple and fast. Show a random number after time
##Settings
- Default:
  + minNum = 1;
  + maxNum = 50;
  + numInterval = 100;
  + limitTime = 5000; /*milisecond*/
  
  *Also you can customize the parameters on the screen
